# The Met · A Curated Walk

二十件镇馆之宝 · 一条路线 · 极简优雅的纽约大都会博物馆导览。

## 这是什么

一个**单文件**的、可在手机和桌面运行的 Met 导览 web app：
- 二层楼层切换（一层 / 二层）
- 每层一张 SVG 平面图（基于公开展厅布局信息原创绘制，非官方地图复制）
- 红铜色虚线路径标明 20 件镇馆之宝的观展顺序，箭头指示方向
- 浏览器 GPS 定位 → 在地图上显示蓝色脉冲点（你的大致位置）
- 点击编号 → 弹出底部抽屉，含英文/中文名称、艺术家、年代、展厅号、背景介绍、欣赏指引
- 抽屉内置「上一件 / 下一件」可顺路线连续浏览

## 一键部署到 Vercel

**方式一（最快，无需命令行）：**
1. 访问 https://vercel.com/new
2. 把这个文件夹拖进上传区（或先推到 GitHub 仓库再 Import）
3. Framework Preset 选 **Other**（纯静态站点）
4. Build Command 留空，Output Directory 留空
5. 点 Deploy

约 30 秒后会拿到一个 `https://xxx.vercel.app` 的链接。

**方式二（命令行，更专业）：**

```bash
npm i -g vercel
cd met-guide
vercel
# 第一次会要你登录，按提示走完即可
# 之后直接 vercel --prod 发布到正式域名
```

## 关于「室内 GPS」的真相

博物馆室内 GPS 信号很弱，浏览器返回的位置精度通常是 10–50 米。要真正做到展厅级精度，需要博物馆部署蓝牙信标（Met 官方 app 就是这么做的）——这是单纯网页无法实现的硬件依赖。

本 app 的处理方式：
- 你在馆内 → 在地图上显示你的估算位置（蓝点），状态栏显示"在馆内 · ±Xm"
- 你在馆外 → 状态栏显示"馆外 · 已定位"，便于你出发前预览路线

## 关于「明天正在陈设」

Met 有 200 万+ 藏品，大部分常设杰作（如丹铎神庙、华盛顿渡河、维米尔、伦勃朗、梵高、莫奈、明轩等）常年在展。这份清单聚焦于**几乎不会撤展的核心镇馆之宝**，能最大化"明天去也能看到"的命中率。

特展（轮换展）建议出发前一晚到 metmuseum.org 官网 "What's On" 页面核对。

## 设计语言

- 字体：Cormorant Garamond（衬线·标题）+ Inter（无衬线·正文）+ Noto Serif SC（中文）
- 配色：米黄纸调底色 + 墨黑文字 + Met 标志性红铜色（accent）
- 动效：路径流动虚线、用户位置脉冲、抽屉滑入、SVG marker hover 反转
